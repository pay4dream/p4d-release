试用 Manual

从安装到生成第一条成片

← 回下载页

1. 安装

  • macOS：打开下载的 .dmg，把 Pay4Dream 拖进「应用程序」。 首次打开右键图标 → 打开，弹窗里再点「打开」。若提示「已损坏」， 终端运行 xattr -dr com.apple.quarantine "/Applications/Pay4Dream.app" 后重试。
  • Windows：双击 .exe 走安装向导。SmartScreen 弹窗点 「更多信息」→「仍要运行」。

2. 首次启动：输入邀请码

第一次打开会要求输入邀请码。把拿到的邀请码粘进去激活即可。 激活信息保存在本机，下次启动不再要求。

3. 填入你自己的 API Key

本版需要你自带模型 Key。进入设置 → 模型 / API Key，填入你的 OpenAI 或 Anthropic Key 并保存。

  • Key 只保存在本机，请求直接发往对应模型厂商，不经第三方中转。
  • 没有 Key？到 OpenAI 或 Anthropic 官网申请后再回来填。

4. 生成第一条成片

  • 新建一个项目，用自然语言描述你想要的内容（主题、风格、时长等）。
  • 按提示一步步生成脚本 / 分镜 / 画面 / 成片，每步都可修改后再继续。
  • 生成结果可在项目里预览和导出。

5. 常见问题

  • 提示 Key 无效 / 额度不足：到模型厂商后台确认 Key 状态和余额。
  • 打不开 / 被系统拦：见第 1 步的右键打开 / SmartScreen 放行。
  • 生成很慢或失败：检查网络能否访问对应模型厂商，稍后重试。

6. 反馈

问题和建议请到 下载页的反馈入口（GitHub Issues）提交， 附系统版本、复现步骤和截图。
